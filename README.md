## ğŸš€ Quick Start

Getting this project up and running is straightforward as it does not require any build tools or server-side dependencies.

### Prerequisites
None! You only need a modern web browser.

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Sehar-1207/EducateMe-online.git
    cd EducateMe-online
    ```

2.  **Open in your browser**
    Simply open the `index.html` file directly in your preferred web browser.
    ```bash
    # For macOS
    open index.html

    # For Windows
    start index.html

    # For Linux
    xdg-open index.html
    ```
    Alternatively, you can drag and drop `index.html` into your browser.
